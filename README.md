## Motivation

One widely shared and battle-tested [Overleaf template](https://www.overleaf.com/project/662939053eb1dc8f0f3c617f) became a strong reference for what _actually works_ in practice.

This project translates that proven, no-nonsense r√©sum√© style into an **interactive web-based editor**, allowing users to:
